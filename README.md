# Dream Catcher
An **ElectronJS** based desktop application originally designed in Windows but portable to MacOS and Linux. 

The features of this application are based on pscyhology and successful mindsets. 

## Features

- [ ] Set **goals**, create **gameplans**, track **progress**, and realize your dreams.

- [ ] Reflect on your values, motivations, and drive with the **journal**. 

- [ ] Schedule **notifications** for tasks, as well as weekly or daily journal entries. 

- [ ] Treat yourself benevolently! Customize healthy **rewards** that give you a much needed break after accomplishing a task. Set bigger rewards for an amazing accomplishment.
